## Quick Start

1. **Clone the repository:**

    ```bash
    git clone https://github.com/BncPntk/test-ciapp.git
    ```

2. **Install dependencies:**

    ```bash
    composer update
    ```

3. **Run migrations:**

    ```bash
    php spark migrate
    ```

4. **Start the development server:**

    ```bash
    php spark serve
    ```
